Classes

InviteCommand

Invites the specified Faction's to the sender's Team.

JoinCommand

Joins the specified Team. Only works if an invite has first been extended.

ObserverCommand

When a player types the command, they get removed from the game and made into an observer.

UnallyCommand

A command that causes a player to leave their Team and be given a solo Team with their Faction as the name.

UninviteCommand

Revokes an invitation sent to a player from the sending player's Team.
